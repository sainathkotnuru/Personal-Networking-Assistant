# 2. Requirement Analysis

## Project Requirement
Develop a user-friendly web application that analyzes event descriptions and produces personalized networking conversation support.

## Functional Requirements
- Accept event name and description
- Extract themes from event description
- Generate conversation starters
- Generate follow-up questions and suggestions
- Verify topics through Wikipedia-based fact checking
- Save chat history locally
- Capture user feedback

## Non-Functional Requirements
- Simple and fast UI
- Modular Python backend
- Testable and maintainable code structure
- Lightweight deployment support
