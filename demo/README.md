# Demo Assets

This folder contains supporting materials for demonstrating the Personalized Networking Assistant.

## Suggested Demo Flow
1. Open the deployed app.
2. Enter an event name such as "Tech Meetup".
3. Paste a short description like "A networking event for software engineers interested in AI and cloud computing."
4. Click "Analyze Event".
5. Click "Generate Conversation Starters".
6. Review the generated themes, conversation starters, follow-up questions, and fact checks.

## Demo Notes
- The app uses theme extraction, conversation generation, and Wikipedia-based fact checking.
- The experience is designed to help users prepare for professional networking conversations.
