# 5. Project Development Phase

## Development Notes
This repository contains the implementation of the Personalized Networking Assistant.

## Main Components
- Streamlit UI: user-side interaction
- FastAPI API: application endpoints
- Services: theme extraction, topic generation, fact checking, history, and feedback
- Storage: JSON files for persistent local data

## Run Instructions
- Install dependencies from requirements.txt
- Run the backend with Uvicorn
- Run the frontend with Streamlit
